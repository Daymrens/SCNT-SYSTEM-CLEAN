# SCNT System - Empty Version

This is a clean, empty version of the SCNT Perfume Inventory System ready for deployment.

## What's Included

- All application source code files
- Empty database (no sample data)
- Default admin account only
- All forms and features ready to use

## Default Login

- **Username:** admin
- **Password:** admin123

**Important:** Change the default password after first login!

## Getting Started

1. Open `PerfumeInventory.csproj` in Visual Studio
2. Build the solution
3. Run the application
4. Login with the default credentials
5. Start adding your own data:
   - Suppliers
   - Customers
   - Resellers
   - Perfume inventory
   - Purchase orders
   - Sales

## System Requirements

- .NET 8.0 or higher
- Windows OS
- Visual Studio 2022 (recommended)

## Features

- Inventory Management
- Point of Sale (POS)
- Customer Management
- Supplier Management
- Reseller Management
- Purchase Orders
- Sales History
- Reports & Analytics
- Dashboard
- Tester Management
- User Management

## Data Persistence

The system automatically saves data to `perfume_inventory_data.json` in the application directory.
